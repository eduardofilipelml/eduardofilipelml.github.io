Data Format Description Language for Space (DFDL4S)

A generic binary data binding library (Java and C++) based on the DFDL standard. Enables reading and writing binary data (e.g., CCSDS Space Packets) with DFDL schema files. Available for Windows, Linux, and Mac OS X platforms.

Space to Ground (S2G) Data Viewer

An extensible software application that allows inspection of CCSDS-compliant data exchanged between satellites and ground segments. Supports Copernicus ESA Mission (S1-S6), Earth Explorers (Aeolus, Biomass, FLEX, Swarm, SMOS, EarthCARE), and MetOp-SG-A/B and MTG satellites.
